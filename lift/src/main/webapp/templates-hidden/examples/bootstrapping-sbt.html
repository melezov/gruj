Using GRUJ as a Simple Build Tool launch launcher

The easiest way to see this in action would be to checkout the GRUJ website from either GitHub Launchpad or and simply run the thing locally.

The web project includes:

  • SBT 0.12.4, with included plugins:
    • SBT eclipse 2.2.0
    • XSBT Web plugin 0.3.0
    • Coffeescripted SBT plugin 0.2.3
    • Less SBT plugin 0.1.10
  • Lift web framework 2.5.1, running on:
    • Jetty 8.1.11.v20130520

Required steps:

Clone the project from Github or download and extract it to some folder. 

user@box:~$ git clone git://github.com/melezov/gruj.git

user@box:~$ cd gruj

… alternatively, branch the project from Launchpad: 

user@box:~$ bzr branch lp:gruj
Branched 39 revision(s).
user@box:~$ cd gruj

Run prepare-workspace.sh (or .bat if on Windoze). This will fire up GRUJ and download SBT 0.11.x launcher from Typesafe repos. Although the launcher download will only take a few seconds, the update process could take a while if you do not have the dependencies in your ivy cache already.
